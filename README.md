## 特性：

1. 提供websocket、命令行窗口、web界面三种方式
2. 单文件可执行，无需安装配置
3. 简洁美观的操作界面

## 使用：

```
下载可执行文件https://github.com/1458790210/RedisCview/releases

不需要windows窗口执行：go build -ldflags "-s -w -H=windowsgui"
```
